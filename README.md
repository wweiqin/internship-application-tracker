# Internship Application Tracker

A web application that helps students organise internship applications, deadlines, recruiter contacts and application outcomes.

## Project Status

🚧 In development

The initial React and TypeScript project structure has been created.

## Problem

Internship seekers commonly track applications through spreadsheets, browser bookmarks and email threads. This makes it difficult to monitor deadlines, follow-ups and application progress.

## Planned Features

- Add, edit and delete applications
- Track application statuses
- Record deadlines and job links
- Store recruiter or contact details
- Search and filter applications
- View application statistics
- Export records as CSV
- Compare job-description keywords with resume skills

## Technology

- React
- TypeScript
- Vite
- SQL
- PostgreSQL or Supabase

## Repository Structure

- `frontend/` — React web application
- `database/` — SQL schema and sample data
- `docs/` — Planning and documentation
- `screenshots/` — Application screenshots

## Local Development

1. Open the frontend folder:

   `cd frontend`

2. Install dependencies:

   `npm install`

3. Start the development server:

   `npm run dev`
