# Internship Application Tracker — Project Plan

## Problem

Students often manage internship applications across spreadsheets, bookmarks and email threads. This makes deadlines, follow-ups and application outcomes difficult to track consistently.

## Proposed Solution

Build a web application that allows students to manage internship applications, monitor deadlines and review their application progress from one dashboard.

## Target Users

- University and polytechnic students
- Fresh graduates
- Internship and job seekers

## Planned Features

### Application Management

- Add, edit and delete applications
- Record company and role details
- Save job listing links
- Record application dates and deadlines
- Add recruiter information and personal notes

### Status Tracking

- Interested
- Applied
- Online Assessment
- Interview
- Offer
- Rejected
- Withdrawn

### Dashboard

- Total applications
- Applications by status
- Upcoming deadlines
- Interview rate
- Offer rate
- Recently updated applications

### Search and Filtering

- Search by company or role
- Filter by application status
- Sort by date, company or deadline

### Future Features

- Job-description keyword matching
- Resume keyword comparison
- CSV export
- Deadline reminders
- User authentication

## Technology

- React
- TypeScript
- Vite
- SQL
- PostgreSQL or Supabase
- GitHub

## Development Approach

The project will begin with a frontend prototype using sample data. A SQL database will then be added for persistent storage and CRUD operations.

## Current Status

Project skeleton and React frontend created.
